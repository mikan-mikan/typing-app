## プロジェクト概要

このプロジェクトは、Webブラウザ上で動作するタイピング練習アプリケーションです。
日本語（漢字・ひらがな）の問題をローマ字入力でタイピングし、スコアを競うことができます。

- **技術スタック**: Next.js 16, React 19, TypeScript, Tailwind CSS v4
- **主要ライブラリ**: wanakana（日本語⇔ローマ字変換）
- **開発ツール**: ESLint, Prettier
- **バージョン**: v0.2.0
- **パッケージ数**: 387個
- **セキュリティ**: 脆弱性 0個
